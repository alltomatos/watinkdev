# Usuários e Filas

Gestão administrativa de acesso e distribuição de atendimento.

## Usuários (`/users`)
- **Perfis**: Admin ou User.
- **Filas**: Associação de quais filas o usuário atende.
- **Configs**: Preferências individuais (Dashboard, Tema).

## Filas (`/queues`)
- **Criação**: Definição de departamentos (ex: Financeiro, Suporte).
- **Cores**: Identificação visual no Chat.
- **Mensagem de Saudação**: Texto automático ao entrar na fila.

## Outros (`/settings`, `/quickAnswers`)
- **Respostas Rápidas**: Atalhos para mensagens frequentes (digite `/`).
- **Configurações**: Parâmetros globais do sistema (ex: Token API, Horários).
- **Swagger**: Visualizador da documentação da API embutido na aplicação (`/swagger`).
